# fashion-store-php-mysql
A full-stack fashion e-commerce web application featuring product listings, cart functionality, and order management. Developed using PHP, MySQL, HTML, CSS, and JavaScript.

## Fashion Store Web Application

This project is a full-stack e-commerce platform developed to simulate real-world online retail operations. It includes product listing, shopping cart functionality, and order management features.

### Technologies Used
- PHP
- MySQL
- HTML, CSS, JavaScript
